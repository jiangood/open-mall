# Open Mall 电商平台

Open Mall 是一个基于 Spring Boot 和 React 开发的开源电商平台，提供完整的 B2B2C 电商解决方案，包括平台管理端、商家端和买家端三大核心模块。

## 技术栈

### 后端
- Java 17
- Spring Boot 3.5.8
- Open Admin 1.0.1

### 前端
- React 19.0.0
- Umi 4.0.0
- Ant Design 5.20.3
- TypeScript 5.0.0

## 快速开始

### 后端启动
1. 确保已安装 JDK 17 或以上版本
2. 确保已安装 Maven
3. 进入项目根目录
4. 执行以下命令启动后端服务：

```bash
mvn spring-boot:run
```

### 前端启动
1. 确保已安装 Node.js 16 或以上版本
2. 进入 web 目录
3. 安装依赖：

```bash
npm install
```

4. 启动前端开发服务器：

```bash
npm run dev
```

5. 构建生产版本：

```bash
npm run build
```

## 功能模块

### 1. 平台管理端

#### 1.1 首页
- 数据看板：展示平台核心数据，包括订单量、销售额、会员数等
- 待办事项：展示需要处理的任务，如店铺入驻审核、商品审核等

#### 1.2 会员管理
- 会员列表：查看和管理所有会员信息
- 会员评价：管理会员对商品的评价
- 积分管理：设置和管理会员积分规则
- 资金流水：查看会员资金变动记录
- 充值管理：管理会员充值记录

#### 1.3 订单管理
- 商品订单：查看和管理所有商品订单
- 虚拟订单：管理虚拟商品订单
- 售后处理：处理会员的退款、退货申请
- 订单投诉：管理会员对订单的投诉
- 收款与退款流水：查看订单的收款和退款记录

#### 1.4 商品管理
- 商品管理：发布、编辑、上下架商品
- 商品审核：审核商家发布的商品
- 分类管理：设置和管理商品分类
- 品牌管理：管理商品品牌
- 规格管理：设置商品规格
- 计量单位管理：管理商品计量单位

#### 1.5 促销管理
- 优惠券：创建和管理优惠券活动
- 秒杀：设置和管理秒杀活动
- 砍价：管理砍价活动
- 拼团：设置和管理拼团活动
- 积分商品：管理积分兑换商品

#### 1.6 店铺管理
- 店铺管理：查看和管理所有店铺
- 入驻审核：审核商家入驻申请
- 结算管理：管理店铺结算
- 店铺对账：与商家进行对账

#### 1.7 运营管理
- 页面装修：装修平台首页和活动页面
- 分销管理：设置和管理分销规则
- 文章管理：发布和管理平台文章
- 意见反馈：管理会员反馈
- 站内信：发送站内消息给会员
- 短信配置：设置短信模板和发送规则

#### 1.8 统计管理
- 会员统计：统计会员增长和活跃度
- 订单统计：分析订单数据和销售趋势
- 流量统计：统计平台访问量和用户行为
- 商品销量统计：分析商品销售数据

#### 1.9 系统设置
- 权限管理：设置管理员权限
- 角色管理：创建和管理管理员角色
- 部门管理：管理平台部门
- 管理员管理：添加和管理平台管理员
- 系统参数：设置平台系统参数
- OSS配置：配置文件存储服务
- 支付配置：设置支付方式
- 物流配置：配置物流方式
- 敏感词管理：设置和管理敏感词

#### 2.1 店铺首页
- 店铺看板：展示店铺核心数据
- 待办事项：展示需要处理的任务
- 公告：查看平台公告

#### 2.2 商品管理
- 商品发布：发布店铺商品

- 订单处理：处理店铺订单

- 投诉处理：处理订单投诉
- 退款/退货申请：处理会员退款、退货申请

#### 2.4 财务管理
- 店铺对账：与平台进行对账
- 结算管理：管理店铺结算
- 发票管理：管理店铺发票

#### 2.5 促销管理
- 优惠券：创建店铺优惠券
- 满减：设置店铺满减活动
- 秒杀：设置店铺秒杀活动
- 拼团：设置店铺拼团活动
- 分销商品：管理店铺分销商品

#### 2.6 统计管理
- 订单统计：分析店铺订单数据

- 商品销量排行：分析店铺商品销售排行

#### 2.7 店铺设置
- 物流配送：设置店铺物流方式
- 自提点：设置店铺自提点
- 店铺信息：编辑店铺基本信息
- PC/移动端装修：装修店铺PC端和移动端页面
- 店员与权限管理：管理店铺店员和权限

### 3. 买家端

#### 3.1 会员功能
- 注册登录：支持手机号注册和登录
- 短信登录：支持手机号短信验证码登录
- 密码重置：通过短信重置密码
- 个人信息管理：编辑个人资料
- 收货地址管理：添加、编辑、删除收货地址
- 我的订单：查看和管理个人订单
- 我的优惠券：查看和使用优惠券
- 我的收藏：管理收藏的商品
- 浏览历史：查看商品浏览历史

#### 3.2 商品功能
- 商品搜索：搜索商品
- 商品分类：按分类浏览商品
- 商品详情：查看商品详细信息
- 商品评价：查看和发表商品评价

#### 3.3 购物车功能
- 添加商品：将商品加入购物车
- 修改数量：调整购物车商品数量
- 选择商品：选择要结算的商品
- 结算：生成订单并结算

#### 3.4 订单功能
- 创建订单：提交购物车商品生成订单
- 支付订单：选择支付方式支付订单
- 查看订单：查看订单详情和状态
- 确认收货：确认收到商品
- 申请售后：申请退款、退货
- 物流查询：查询订单物流信息

#### 3.5 促销功能
- 领取优惠券：领取平台和店铺优惠券
- 参加活动：参加秒杀、拼团等活动
- 积分兑换：使用积分兑换商品

#### 3.6 支付功能
- 在线支付：支持多种支付方式
- 充值：为账户充值

#### 3.7 其他功能
- 意见反馈：提交意见和建议
- 联系客服：与平台客服沟通
- 站内信：查看平台发送的消息

## 开发指南

### 后端开发
1. 后端代码位于 `src/main/java/com/example/` 目录
2. 配置文件位于 `src/main/resources/` 目录
3. 使用 Maven 进行依赖管理和构建

### 前端开发
1. 前端代码位于 `web/src/` 目录
2. 配置文件位于 `web/config/` 目录
3. 使用 npm 进行依赖管理
4. 使用 Umi 进行前端路由和构建
